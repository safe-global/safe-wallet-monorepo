# Module Inventory

Snapshot of where to find canonical examples ("copy this") and known
legacy / special-case areas ("don't copy this") in the monorepo, used
by the convention skill to ground rule generalization.

This is a maintenance document. Normal coding agents should read
[../../conventions/project-structure.md](../../conventions/project-structure.md), not this file.

## Workspaces

| Path               | Workspace             | Stack                     | Notes                                                |
| ------------------ | --------------------- | ------------------------- | ---------------------------------------------------- |
| `apps/web`         | `@safe-global/web`    | Next.js + MUI + Tailwind  | Primary web app; lazy-loaded feature folders         |
| `apps/mobile`      | `@safe-global/mobile` | Expo + Tamagui            | iOS and Android client                               |
| `packages/store`   | `@safe-global/store`  | Redux Toolkit + RTK Query | Shared store, AUTO_GENERATED CGW types               |
| `packages/utils`   | `@safe-global/utils`  | Pure TS                   | Shared utilities; consumed by web and mobile         |
| `packages/theme`   | `@safe-global/theme`  | Pure TS                   | Tokens for MUI + Tamagui generators                  |
| `config/test/msw`  | n/a                   | MSW                       | Static CGW response fixtures                         |
| `apps/web/cypress` | n/a                   | Cypress                   | E2E suite (smoke / visual / regression / happy path) |

## Canonical "Copy This" Examples

| Concern                                        | Reference                                                                                                                    |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Lazy-loaded feature with hooks + components    | `apps/web/src/features/__core__/createFeatureHandle.ts`, plus any feature whose `feature.ts` matches the flat-export pattern |
| RTK Query persistence with cache-key filtering | endpoints under `packages/store/src/gateway/`                                                                                |
| Test setup wrapper with providers              | `apps/web/src/tests/test-utils.tsx`                                                                                          |
| Builder helpers                                | `apps/web/src/tests/Builder.ts` and `apps/web/src/tests/builders/*`                                                          |
| Cypress feature-flag helper                    | `apps/web/cypress/e2e/pages/main.page.js` `injectChainFeature`                                                               |
| Address comparison utility                     | `packages/utils/src/utils/addresses.ts` `sameAddress`                                                                        |
| Safe version normalization                     | `getCleanSafeVersion` in `packages/utils/src/utils`                                                                          |
| Theme generation                               | `packages/theme/src/index.ts` `generateMuiTheme`, `generateTamaguiTokens`                                                    |

## Legacy / Special-Case Areas

These deserve extra caution. They are not patterns to copy.

| Path                                                    | Caveat                                              |
| ------------------------------------------------------- | --------------------------------------------------- |
| `apps/web/src/components/ui/*`                          | Generated by shadcn; edits in place are overwritten |
| `packages/store/src/gateway/AUTO_GENERATED/`            | Schema-generated; never hand-edit                   |
| `packages/utils/src/types/contracts/`                   | Generated from contract ABIs                        |
| Long inline Tailwind class strings in legacy components | Pending CSS module extraction                       |
| `apps/mobile/assets/wallet-icons/*.ts` (historical)     | Past offender for embedded data URIs                |
